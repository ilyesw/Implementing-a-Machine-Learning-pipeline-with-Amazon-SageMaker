# üá¨üáß English Version

This repository contains a series of practical labs demonstrating the end-to-end process of building, training, and deploying machine learning models using Amazon SageMaker.

## Project Structure

The project is divided into 5 labs, each covering a key step of the ML pipeline:

1.  **Lab 3.1 - Amazon SageMaker and Importing Data**
    - Introduction to the Amazon SageMaker environment.
    - Configuration and data import for analysis.

2.  **Lab 3.2 - Exploring Data**
    - Exploratory Data Analysis (EDA) techniques.
    - Visualization and understanding of datasets.

3.  **Lab 3.3 - Encoding Categorical Data**
    - Data preprocessing.
    - Encoding techniques for categorical variables (One-Hot Encoding, Label Encoding, etc.).

4.  **Lab 3.4 - Training a Machine Learning Model**
    - Algorithm selection and configuration.
    - Model training process on prepared data.

5.  **Lab 3.5 - Deploying and Using a Machine Learning Model**
    - Deployment of the trained model.
    - Creation of endpoints and inference.

## Prerequisites

- AWS account with access to Amazon SageMaker.
- Basic knowledge of Python and Machine Learning.
- Jupyter Notebooks.
- Basic knowledge of AWS.

## Installation and Usage

1.  Clone this repository.
2.  Navigate to the desired lab folder.
3.  Open the `.ipynb` files (Jupyter Notebooks) in your SageMaker or local Jupyter environment.
4.  Follow the instructions in each notebook to execute the code.
